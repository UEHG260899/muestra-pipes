Pipes incluidos en Angular - Usualmente en el CommonModule.

Si se desea convertir un String a mayusculas se usa la variable seguido de | uppercase. Ejemplo:

  • String sin pipe: {{ nombreLower }}
  • String con pipe: {{ nombreLower | uppercase }}

Si se desea convertir un String a minusculas se usa la variable seguido de | lowercase. Ejemplo:

  • String sin pipe: {{ nombreUpper }}
  • String con pipe: {{ nombreUpper | lowercase }}

Si se desea capitalizar un String a minusculas se usa la variable seguido de | titlecase. Ejemplo:

  • String sin pipe: {{ nombreCompleto }}
  • String con pipe: {{ nombreCompleto | titlecase }}
